General HTML Info

HTML Itself

HTML is what a browser reads to display a page. Contains refernces to other objects on the host server. HTML stands for HyperText Markup Language. HTML is the main document of the web

HTML has:

-readable text that describes the page's content
-markup tags denote what the page will look like
-references to other files
-links to other HTML pages

Technical Info

Tags

Tags describe what the content of a page will look like. The general format the content between the opening and closing tags. An Element is the complete object, like text with a body tag. A Void Tag does not contain content so there is no closing tag. For example, line breaks are void tags because there is no closing tag.

WhiteSpace

Spaces and indenation in HTML code do not transfer to the displayed page. Tags need to be used to indicate spacing and indents on the displayed page. Tags like br, p, span, and div create spaces and line breaks on the displayed page. Some of these tags can denote the content as being in a new element.

Inline and Block

-inline:affects text content without creating new elements
-block:new collection of content, certain tags can create new blocks like the paragraph tag.

Document Structure

1.HTML declaration, indicationg that the file is in html.
2.Head: containing the title and metadata and things like CSS.
3.Body: content of the page.
